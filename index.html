Drawers Installation Guide

Step 1: Download the Extension

  1. Open the Google Drive folder
  2. Click the Download button
  3. Wait for Drawers.zip to download

Step 2: Extract the Files

  1. Locate Drawers.zip in your Downloads folder
  2. Double-click (Mac) or right-click and select "Extract All" (Windows)
  3. Move the extracted Drawers folder to your Desktop
Important: Don't delete this folder after installation. Chrome needs it to run the extension.

Step 3: Open Chrome Extensions

  1. Open Google Chrome
  2. Type chrome://extensions/ in the address bar
  3. Press Enter

Step 4: Enable Developer Mode

  1. Find the "Developer mode" toggle in the top-right corner
  2. Click to turn it ON (it will turn blue)
  3. New buttons will appear: "Load unpacked", "Pack extension", and "Update"

Step 5: Load the Extension

  1. Click "Load unpacked"
  2. Navigate to your Desktop (or wherever you saved the Drawers folder)
  3. Select the Drawers folder
  4. Click "Select Folder" (Mac) or "Select" (Windows)
Note: Select the Drawers folder itself, not a file inside it.

Step 6: Pin the Extension

  1. Look for the Drawers card in your extensions list
  2. Click the puzzle piece icon (üß©) in your Chrome toolbar
  3. Find "Drawers" and click the pin icon to keep it visible

Step 7: Start Using Drawers

  1. Click the Drawers icon in your toolbar
  2. Click "+ Add Drawer" to create your first drawer
  3. Name it and click "Create"
  4. Navigate to any webpage and click "+ Add Card" to save it
Tip: The URL, title, and description fields will auto-fill from the current page.

Setup Complete

Drawers is now installed and ready to use.

Quick Reference:

  • Create drawers to organize resources by topic
  • Add cards to save links with descriptions
  • Use tags to filter and organize cards
  • Search across all drawers to find resources
  • Download/upload drawers to share with others
